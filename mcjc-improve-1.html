「MCBE Server Dev Japan Community」(仮名) への変更、「MCBE Server Japan Community」の新設

現在のサーバーは開発内容メインで、名前とあっていないので、サーバーを移行し、現在のサーバーを開発専用、移行先のサーバーを雑談用とすることによって、現在の内容を残したまま、多くのユーザーが「MCBE Server Japan Community」に参加しやすくなると思います。

要するに、「開発者向け - For Developer」の内容「広告 - For Advertisement」のプラグイン宣伝チャンネルを残して、雑談等を別サーバーに移すということです。

Lobi「鯖主ティアクターズ」との連携

Lobiとの両立として、「鯖主ティアクターズ」をコミュニティーのLobiグループとして利用します。

禁止事項の改定

禁止事項を以下のように変更します。
変更は2018/02/24 18:00より適用します。

現行

  • 他人が不愉快に思う発言や返信、メディアの投稿
  • 過度な連続投稿
  • その他常識が守られていない行為

改定後

  • 他人が不愉快に思う発言や返信、メディアの投稿
  • 過度な連続投稿
  • 不適切なコンテンツへの誘導、もしくは使用を推奨する投稿
  • その他公序良俗に反する行為
